## Get Started

`git clone` 後、以下のコマンドを実行します。

```shell
$ make bs
```

環境が整っているかどうかを確認する場合は以下のコマンドを実行します。

```shell
$ make doctor
```

## 記事の作成

```shell
$ bun new:article
```

## プレビュー

```shell
$ bun preview
```
